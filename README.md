Project Summary

This project is a backend API for a restaurant review website.
It allows users to:

Create new restaurants and reviews

Read restaurant details and reviews

Update restaurant information and reviews

Delete restaurants or reviews

Each restaurant entry includes:

Name

Address / Location

Cuisine type

Number of stars / ratings

Reviews

The API is designed to manage data efficiently and provide a solid foundation for any frontend or mobile app integration.

Tech Stack
Component	Technology
Backend	Node.js, Express.js
Database	MongoDB (or MySQL/PostgreSQL)
Environment	dotenv
Development Tool	nodemo
